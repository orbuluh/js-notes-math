# Math + Plotly Demo

This page shows inline math like $e^{i\pi} + 1 = 0$ and a display equation:

$$
\int_0^1 x^2 \, dx = \frac{1}{3}
$$

## Python Code Block

```python
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)
print(y[:5])
```
