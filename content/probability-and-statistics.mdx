### 1. 期望值 (Expected Value, $E[X]$)

**期望值**是隨機變數在長期重複實驗下的「平均」表現，也被稱為**均值 (Mean)**。

*   **定義與公式**：
    *   **離散型隨機變數**：將所有可能的數值乘以其發生的機率後加總。公式為 $E[X] = \sum x p(x)$。
    *   **連續型隨機變數**：數值乘以機率密度函數後對整個區間進行積分。公式為 $E[X] = \int_{-\infty}^{\infty} x f(x) dx$。
*   **直觀理解**：期望值可以被視為機率分布的**重心 (Center of Gravity)**。例如，擲一粒公正骰子的期望值是 3.5，這代表如果你擲很多次，平均值會趨近於這個數。
*   **重要性質**：和的期望值等於期望值的和，即 $E[X+Y] = E[X] + E[Y]$，此性質不要求兩變數必須獨立。

### 2. 變異數 (Variance, $Var(X)$ 或 $\sigma^2$)

**變異數**衡量隨機變數數值的**分散程度**或**波動大小**。

*   **定義**：它是隨機變數與其期望值（均值）之間距離平方的期望值。公式為 $Var(X) = E[(X - E[X])^2]$。
*   **計算公式**：實務上常用的簡化公式為「平方的期望值減去期望值的平方」，即 $Var(X) = E[X^2] - (E[X])^2$。
*   **直觀理解**：變異數越大，代表數據相對於均值越分散，不確定性也越高。它始終為非負數。

### 3. 標準差 (Standard Deviation, $std(X)$ 或 $\sigma$)

**標準差**是變異數的正平方根。

*   **公式**：$SD(X) = \sqrt{Var(X)}$。
*   **直觀理解與用途**：
    *   **回歸原始單位**：變異數因為經過平方計算，單位會變成原始單位的平方（例如「 pets 平方」）；標準差則將尺度**縮回原始單位**（例如「幾隻 pet」），使其更具意義。
    *   **衡量分佈**：在常態分布中，標準差定義了曲線的寬度。約 68% 的數據落在均值正負 1 個標準差內，95% 落在 2 個標準差內。

### 4. 共變異數 (Covariance, $Cov(X, Y)$)

**共變異數**衡量兩個隨機變數之間的**線性關係方向**。

*   **定義與公式**：$Cov(X, Y) = E[(X - E[X])(Y - E[Y])]$。
*   **計算公式**：$Cov(X, Y) = E[XY] - E[X]E[Y]$。
*   **直觀理解**：
    *   如果 $Cov(X, Y) > 0$，兩者傾向於同向變動（一增則另一增）。
    *   如果 $Cov(X, Y) < 0$，兩者傾向於反向變動。
    *   如果 $Cov(X, Y) = 0$，則稱兩者為**不相關 (Uncorrelated)**。**獨立**的變數其共變異數必為 0，但反之則不一定成立。

### 5. 相關係數 (Correlation Coefficient, $cor(X, Y)$ 或 $\rho$)

**相關係數**是經過**標準化（歸一化）**後的共變異數，用於精確衡量兩變數間線性關係的**強度**與**方向**。

*   **定義與公式**：將共變異數除以兩個變數各自標準差的乘積。公式為 $\rho(X, Y) = \frac{Cov(X, Y)}{\sigma_X \sigma_Y}$。
*   **特點**：
    *   **數值範圍**：其值介於 **-1 與 1** 之間。
    *   **無單位**：它是一個無量綱的數值，不會受到數據縮放（例如從公尺改為公分）的影響。
    *   **強度指標**：越接近 1 或 -1 代表線性關係越強；接近 0 則代表缺乏線性關係。
    *   **注意**：相關並不等於因果關係。

### 總結對照表

| 概念 | 英文 (符號) | 核心功能 | 主要公式 |
| :--- | :--- | :--- | :--- |
| **期望值** | Expected Value ($E[X]$) | 找中心點（平均） | $\sum x p(x)$ 或 $\int x f(x) dx$ |
| **變異數** | Variance ($Var(X)$) | 量化分散程度 | $E[X^2] - (E[X])^2$ |
| **標準差** | Standard Deviation ($\sigma$) | 回歸原始單位的分散度 | $\sqrt{Var(X)}$ |
| **共變異數** | Covariance ($Cov(X, Y)$) | 看兩變數變動方向 | $E[XY] - E[X]E[Y]$ |
| **相關係數** | Correlation ($cor(X, Y)$) | 看兩變數線性關係強弱 | $Cov(X, Y) / (\sigma_X \sigma_Y)$ |


---

這些數學公式之所以能精確表達直觀理解，是因為它們將物理世界的幾何與力學概念（如重心、慣性）轉化為機率語言，並透過代數運算解決了實際測量中的抵銷與尺度問題。以下是詳細的對應分析：

### 1. 期望值 $E[X]$：為何是「重心」與「長期平均」？
*   **幾何重心（Center of Gravity）**：在物理學中，如果有一根無重量的槓桿，在不同位置 $x_i$ 放上重量 $p(x_i)$，槓桿達到平衡的點就是重心。數學公式 $\sum x p(x)$ 正好計算了這個**平衡點**，確保該點兩側的力矩（距離乘以機率）之和為零。
*   **頻率詮釋（Frequency Interpretation）**：根據**大數法則**，如果實驗重複無限次，某個數值出現的比例會趨於其機率 $P(E)$。因此，將每個數值乘以其發生的機率再加總，本質上就是在計算所有可能結果在**長期重複下的平均報酬**。

### 2. 變異數 $Var(X)$：為何要「平方」？
*   **避免正負抵銷**：如果直接計算數據與均值的平均距離 $E[X - E[X]]$，正偏差與負偏差會互相抵銷，結果永遠是 0。
*   **放大極端偏差**：使用平方 $E[(X - \mu)^2]$ 可以確保所有距離皆為正數，且能**放大較大的偏差**，這在風險管理中非常重要，因為大的波動通常代表更高的不確定性。
*   **物理慣性（Moment of Inertia）**：在力學中，變異數對應於**轉動慣量**，代表物體抵抗運動狀態改變的能力。在統計中，這代表數據分佈的穩定程度；變異數越大，分佈越「重」，越難以預測。

### 3. 標準差 $std(X)$：為何要「開根號」？
*   **回歸原始尺度**：變異數因為經過平方計算，其單位會變成原始單位的平方（例如「人數的平方」），這在直觀上難以理解。
*   **還原物理意義**：透過開根號 $\sqrt{Var(X)}$，我們能將波動大小**還原回原始單位**（例如「人數」），使其能直接與均值進行比較或進行標準化計算。

### 4. 共變異數 $Cov(X, Y)$：為何能看「變動方向」？
*   **乘積的正負號**：公式 $E[(X - \mu_X)(Y - \mu_Y)]$ 的核心在於括號內的乘積。
    *   如果 $X$ 高於均值時，$Y$ 也通常高於均值，則兩個括號皆為正，乘積為正。
    *   如果兩者趨向於反向變動，乘積則為負。
*   **線性趨勢**：這種結構捕捉了兩者是否在**線性空間中共同演進**。如果兩者完全獨立，其共變異數必為 0。

### 5. 相關係數 $cor(X, Y)$：為何要「標準化」？
*   **消除尺度的影響**：共變異數會隨數據的單位（如公分改為公尺）而改變大小，這讓人難以判斷關係到底有多強。
*   **純粹的相關強度**：將共變異數除以兩個隨機變數各自的標準差 $\sigma_X \sigma_Y$，本質上是將數據**標準化（歸一化）**，剔除了單位的干擾。
*   **數學界限**：根據柯西-施瓦茨不等式，這種縮放確保了結果必定落在 **-1 到 1** 之間。這讓不同背景、不同單位的兩組數據（例如「身高」與「體重」）之間的線性強度具備了**可比性**。

### 總結
這些公式並非憑空而來，而是為了量化現實世界的物理性質：**期望值定義了「在哪裡」**，**變異數定義了「多散亂」**，而**相關性則定義了「如何互相牽引」**。透過數學分析，我們能將模糊的直觀感受轉化為計算機可運算的精確數字。

在機率論與統計學的實際應用中，理解隨機變數的行為通常依賴於兩種核心函數：**機率密度函數 (PDF)** 與 **累積分佈函數 (CDF)**。雖然許多古典模型基於**高斯（常態）分佈**的假設，但現實世界的金融報酬往往展現出更複雜的特性，如**厚尾 (Heavy tails)** 與**偏態 (Skewness)**，這使得從業者必須轉向使用 **Student's t-分布** 或 **冪律模型 (Power-law models)**。

以下根據參考資料為您詳細說明這些概念：

### 1. 核心函數：PDF 與 CDF

*   **機率密度函數 (Probability Density Function, PDF)**：
    *   對於連續隨機變數，PDF $f(x)$ 描述了數值落在某個特定區間的「密度」。
    *   **重點**：在連續分佈中，隨機變數取某個精確值的機率為 0。
    *   **計算**：落在區間 $[a, b]$ 的機率是透過對 PDF 進行積分獲得的，即 $P\{a < X < b\} = \int_{a}^{b} f(x) dx$。
*   **累積分佈函數 (Cumulative Distribution Function, CDF)**：
    *   CDF $F(x)$ 定義為隨機變數 $X$ 小於或等於某個值 $x$ 的機率，即 $F(x) = P\{X \leq x\}$。
    *   **特性**：它是非遞減函數，範圍介於 0 到 1 之間。
    *   **關係**：PDF 實際上是 CDF 的導數，即 $f(x) = \frac{d}{dx} F(x)$。

### 2. 高斯分佈（常態分佈）及其局限性

*   **定義**：常態分佈是一種對稱的鐘形曲線，由均值 ($\mu$) 和標準差 ($\sigma$) 決定。
*   **流行原因**：**中央極限定理 (CLT)** 指出，許多獨立隨機變數的平均值在樣本量增加時會趨向於常態分佈。這使得它成為建模學生分數、身高或測量誤差的標準工具。
*   **局限性（現實挑戰）**：
    *   **厚尾 (Heavy Tails)**：現實世界的金融數據（如 OMXS30 指數）顯示，極端事件發生的頻率遠高於高斯模型的預測。
    *   **偏態 (Skewness/Asymmetry)**：金融報酬常展現出**損益不對稱性**，即損失的幅度往往比收益更大，且持續時間更短。

### 3. 進階模型：處理厚尾與不確定性

當數據不符合常態分佈的假設時，從業者會改用以下模型：

#### **Student's t-分布 (Student’s t-distribution)**
*   **特性**：其形狀與常態分佈相似，但擁有**更肥厚的尾部 (Fatter tails)**。
*   **應用**：當樣本量較小或母體變異數未知時，t-分布能更好地反映不確定性。在金融研究中，它常被用來替代常態分佈以對應資產報酬中的「厚尾」現象。

#### **冪律模型與 Pareto 分佈 (Power-law & Pareto)**
*   **冪律 (Power Law)**：指某個數值的機率與其大小的冪成反比，例如 $PMF(x) \propto 1/x^\alpha$。
*   **Pareto 分佈**：這是一種典型的冪律分佈，其尾部比指數分佈更厚。
*   **應用場景**：
    *   描述人口中的家庭收入或財富分佈。
    *   網路流量文件的大小或超級電腦完成任務所需的時間。
    *   自然語言中的單字出現頻率（即 **Zipf 定律**）。

### 總結比較

| 特性 | 高斯 (常態) 分佈 | Student's t-分布 | 冪律 / Pareto 分佈 |
| :--- | :--- | :--- | :--- |
| **形狀** | 對稱鐘形 | 對稱，但尾部較厚 | 極度偏斜，長尾 |
| **主要用途** | 建模自然現象 (身高、誤差) | 小樣本推論、金融厚尾建模 | 建模財富、流量、稀有極端事件 |
| **尾部風險** | 忽略極端風險（尾部迅速降至 0） | 捕捉部分極端風險 | 強調極端風險（厚尾特性） |

在實際工作中，從業者通常會透過 **Q-Q 圖 (Quantile-Quantile plot)** 來視覺化檢測數據是否符合常態分佈，若發現觀察點偏離 45 度對角線，則代表存在厚尾或偏態，需考慮使用上述替代模型。

**貝氏統計 (Bayesian Statistics)** 與傳統的頻率論方法不同，它不假設參數（如資產報酬率）是一個唯一且未知的固定常數，而是將參數視為隨機變數，並假設其屬於某種**機率分佈**。在金融投資領域，這種框架（如 Black-Litterman 模型）允許投資者透過貝氏定理，將主觀見解與市場先驗資訊結合，生成更新後的報酬估計。

以下是根據參考資料對此過程的詳細數學與邏輯解釋：

### 1. 先驗分佈 (Prior Distribution)：市場均衡的起始點
貝氏模型的第一步是設定**先驗分佈** $P(H)$，這代表投資者在觀察到新證據或加入主觀見解之前的初始信念。
*   **定義**：這是一個「不依賴於當前觀測數據」的分佈，通常基於直覺、歷史經驗或領域專業知識。
*   **市場應用**：在投資模型中，這通常對應於**市場均衡 (Market Equilibrium)** 的狀態，作為推論的基準起點。

### 2. 似然函數 (Likelihood Function)：主觀見解或新證據
當投資者獲得新資訊（例如對特定資產走勢的主觀見解或最新的市場數據）時，這被視為**證據 (Evidence)** 或數據 $D$。
*   **功能**：似然函數 $P(D|H)$ 描述了在假設（參數分佈）成立的前提下，觀察到這些特定見解或數據的機率。
*   **作用**：它作為一個「修正項」，準備用來調整最初的先驗信念。

### 3. 後驗分佈 (Posterior Distribution)：更新後的報酬估計
貝氏定理的核心在於**歷時性詮釋 (Diachronic Interpretation)**，即隨著時間推移，機率會隨著新數據的加入而更新。
*   **計算過程**：透過將先驗機率乘以似然函數，再除以歸一化常數，得到**後驗分佈** $P(H|D)$。
*   **數學表達**：這是一個「受數據約束」的分佈，它精確地展示了初始意見（先驗）應如何根據新證據（見解）進行修改。
*   **結果**：最終生成的後驗報酬估計，實質上是市場先驗資訊與投資者見解之間的**加權權衡**。

### 4. 貝氏推論的動態特質
貝氏框架在處理投資預測時具有以下優勢：
*   **持續更新**：貝氏推理是一個自然演進的過程。當前的後驗分佈可以作為下一個分析階段的先驗分佈，隨著新見解的不斷湧入，投資者可以持續精煉其預測。
*   **處理不確定性**：貝氏模型不只給出一個預測數值，而是給出整個分佈。這讓投資者能計算**信賴區間（Bayesian Credible Intervals）**，直接衡量預測報酬率落在特定範圍內的信心程度。
*   **對抗先驗偏誤**：雖然先驗分佈具有主觀性，但參考資料指出，當數據量（或見解的強度）足夠大時，不同的初始先驗會趨於收斂，這種現象稱為「淹沒先驗 (Swamping the priors)」。

總結來說，貝氏統計提供了一種系統化的數學地圖，將**主觀信念（見解）**與**客觀背景（市場均衡）**整合在一起，從而產生一個既反映市場結構又包含投資者獨特判斷的**後驗報酬估計**。
